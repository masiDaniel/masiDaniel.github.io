Project Information

This project was created from the need of having a clear and credible structure in the renting industry especially for campus students

The application is built on an intresting tech stack that is

  • frontend built on flutter
  • backend build on django and a bit of php

    • The main feature that is unique to this project is the ability of the users to rate and give actuall real time comments

    Key Features

    Landing page

    This feature enable new users to sign up and teh already existing useres to log in. It also gives an overview of whay the various stakeholders can expect from the application.

    Property Listing

    This feature enable the landlords to be able to showcase their houses to the target audience which is teh students. The students also will be able to view the houses.

    Profile management

    the user is able to get a profile picture update and can get an overview of how many houses he has been in through the platform. it also show their rank as a tenant which is only visiible to them and the landlords.
